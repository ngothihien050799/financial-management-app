✨ Phương pháp 1: Auto-Create Shortcut (DỄ NHẤT) 🌟

  1. Mở thư mục dự án: C:\ADMIN\Code CPC1HN\testai
  2. Tìm file: create-desktop-shortcut.bat
  3. Double-click vào file này
  4. Một cửa sổ Command Prompt sẽ mở và tự động tạo shortcut
  5. Nhấn Enter khi hoàn tất
  6. Kiểm tra Desktop - Shortcut "💰 Financial Management" sẽ xuất hiện!
✓ Hoàn tất! Shortcut đã sẵn sàng trên Desktop. Double-click để khởi động ứng dụng.
⭐ DỄ - Không cần Admin

⚡ Phương pháp 2: PowerShell Script

  1. Mở PowerShell chạy như Admin
  2. Chuyển đến thư mục dự án:
cd C:\ADMIN\Code CPC1HN\testai
  1. Chạy script:
.\create-shortcut.ps1
  1. Chờ thông báo thành công
  2. Shortcut sẽ xuất hiện trên Desktop
💡 Tip: Nếu gặp lỗi "cannot be loaded because running scripts is disabled", chạy:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
⚡ TRUNG BÌNH - Yêu cầu Admin

🖱️ Phương pháp 3: Send To Desktop

  1. Tìm file: start-app.bat trong thư mục dự án
  2. Nhấp chuột phải vào file
  3. Chọn: Send to → Desktop (create shortcut)
  4. Shortcut sẽ tự động xuất hiện trên Desktop
✓ Shortcut tạo xong! Sau đó có thể đổi tên và icon.
⭐ DỄ - 1 cú click

🔧 Phương pháp 4: Manual - Tạo Shortcut Thủ Công

  1. Nhấp chuột phải trên Desktop (vùng trống)
  2. Chọn: New → Shortcut
  3. Paste đường dẫn sau:
cmd.exe /c "C:\ADMIN\Code CPC1HN\testai\start-app.bat"
  1. Nhấp Next
  2. Đặt tên: 💰 Financial Management
  3. Nhấp Finish
  4. (Tùy chọn) Nhấp chuột phải shortcut → Properties → Change Icon
⚡ TRUNG BÌNH - Nhiều bước

💡 Mẹo & Thủ Thuật

📌 Pin to Start Menu Sau khi tạo shortcut, bạn có thể:
1. Nhấp chuột phải shortcut
2. Chọn "Pin to Start"
3. Shortcut sẽ xuất hiện trong Start Menu cho lần sau
⌨️ Keyboard Shortcut Bạn có thể đặt phím tắt:
1. Nhấp chuột phải shortcut → Properties
2. Tìm trường "Shortcut key"
3. Nhập tổ hợp phím (ví dụ: Ctrl+Alt+M)
4. Nhấp Apply rồi OK
⚠️ Lưu ý quan trọng Đảm bảo file start-app.bat không bị di chuyển hoặc xóa, nếu không shortcut sẽ không hoạt động.

🚀 Cách sử dụng Shortcut

  1. Tìm shortcut "💰 Financial Management" trên Desktop
  2. Double-click shortcut
  3. Cửa sổ Command sẽ mở và khởi động server
  4. Browser sẽ tự động mở ứng dụng tại: http://localhost:5173
  5. Bắt đầu sử dụng ứng dụng quản lý tài chính!
✓ Hoàn thành! Bây giờ bạn có thể khởi động ứng dụng chỉ bằng một cú double-click trên Desktop. 🎉
