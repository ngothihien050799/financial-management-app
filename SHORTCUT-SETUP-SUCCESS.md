## 🎨 Customize Your Shortcut

### Change Icon

1. Right-click shortcut → Properties
2. Click "Change Icon..."
3. Choose a different icon
4. Click OK

### Add Keyboard Shortcut (e.g., Ctrl+Alt+M)

1. Right-click shortcut → Properties
2. Click in "Shortcut key" field
3. Press desired keys (e.g., Ctrl+Alt+M)
4. Click Apply → OK
5. Now you can launch with keyboard!

### Pin to Start Menu

1. Right-click shortcut
2. "Pin to Start"
3. Find in Start Menu next time

### Pin to Taskbar

1. Right-click shortcut
2. "Pin to taskbar"
3. Appears in taskbar for instant access
